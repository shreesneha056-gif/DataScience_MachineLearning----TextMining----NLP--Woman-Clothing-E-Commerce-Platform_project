---

## 📌 Problem Statement
E-commerce businesses receive thousands of unstructured customer reviews daily. This project extracts actionable insights from review text using NLP techniques to understand customer sentiment and discover key product themes.

## 🎯 Key Analyses
| Analysis | Technique |
|----------|-----------|
| Sentiment Analysis | TextBlob (Polarity & Subjectivity) |
| Topic Modeling | Latent Dirichlet Allocation (LDA) |
| Exploratory Text Analysis | Word frequency, N-grams, Word clouds |
| Text Preprocessing | Tokenization, Stopword removal, Lemmatization |

## 🛠️ Tech Stack
- **Language:** Python
- **NLP:** NLTK, TextBlob, Gensim (LDA)
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn, WordCloud
- **Environment:** Jupyter Notebook

## 📊 Pipeline Overview
1. **Data Loading** — Load and explore review dataset
2. **Text Preprocessing** — Clean, tokenize, remove stopwords, lemmatize
3. **Exploratory Text Analysis** — Word frequencies, N-grams, word clouds
4. **Sentiment Analysis** — TextBlob polarity and subjectivity scores
5. **Topic Modeling** — LDA to extract hidden product/review themes
6. **Insights** — Business recommendations from text patterns
